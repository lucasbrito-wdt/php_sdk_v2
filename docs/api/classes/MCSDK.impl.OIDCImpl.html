startAuthentication()

startAuthentication(\MCSDK\discovery\DiscoveryResponse  $discoveryResult, string  $redirectURI, string  $state, string  $nonce, string  $scope, integer  $maxAge, integer  $acrValues, string  $encryptedMSISDN, \MCSDK\oidc\AuthenticationOptions  $specifiedOptions, \MCSDK\oidc\IStartAuthenticationCallback  $callback)

Start the authentication comms via rest with the operator

This will form an authentication/authorization request and initiate the request.

Parameters

\MCSDK\discovery\DiscoveryResponse $discoveryResult

the result from start discovery request
string $redirectURI

On completion or error where the result information is sent using a HTTP 302 redirect. (Required).
string $state

Application specified unique scope value.
string $nonce

Application specified nonce value. (Required).
string $scope

Requested scope. (Optional). If omitted or empty defaults to the value "openid".
integer $maxAge

Requested maximum time in seconds since last user authentication. (Optional). If omitted or empty defaults to the value 3600.
integer $acrValues

Requested Authentication Context class Reference (Optional). If omitted or empty defaults to the value "2" .
string $encryptedMSISDN

Encrypted MSISDN for user if returned from discovery service. (Optional).
\MCSDK\oidc\AuthenticationOptions $specifiedOptions
\MCSDK\oidc\IStartAuthenticationCallback $callback

Used to capture the response. (Required).
