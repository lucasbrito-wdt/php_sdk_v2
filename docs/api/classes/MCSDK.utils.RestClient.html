callRestEndPoint()

callRestEndPoint(\Zend\Http\Client  $context, string  $uri, string  $httpPath, array  $params, integer  $timeout, array  $currentCookies = null) : \MCSDK\utils\RestResponse

Make the specified request with the specified client context.

The specified cookies will be added to the request. A request will be aborted if it exceeds the specified timeout. Non Json responses are converted and thrown as RestExceptions.

Ensures that all closable resources are closed.

Parameters

\Zend\Http\Client $context

the zend rest client
string $uri

the uri to use when send requests to GSMA
string $httpPath

the rest path string for instance discovery/test/
array $params

the GET/POST parameters to include in the request
integer $timeout

the request timeout in milliseconds
array $currentCookies

the cookies to send to and from GSMA api

Throws

\Exception

when the rest request has failed in some way
